Helper Classes Reference

Display

  • .hidden - Hide element
  • .flex - Flex container
  • .position-relative - Relative positioning
  • .position-absolute - Absolute positioning
  • .position-fixed - Fixed positioning
  • .position-sticky - Sticky positioning
  • .position-static - Static positioning
  • .position-unset - Unset positioning
  • .item-top-right - Top right position
  • .item-top-left - Top left position
  • .item-bottom-right - Bottom right position
  • .item-bottom-left - Bottom left position

Flex

  • .items-center - Center items vertically
  • .justify-between - Space between items

Text

  • .text-center - Center text
  • .text-left - Left align text
  • .text-right - Right align text

Float

  • .float-left - Float left
  • .float-right - Float right

Clearfix

  • .clearfix - Remove floats

Responsive

  • .sm-hidden - Hide on small screens
  • .md-hidden - Hide on medium screens
  • .lg-hidden - Hide on large screens

Spacing

  • .m-{1-5} - Margin all sides
  • .mx-{1-5} - Horizontal margin
  • .my-{1-5} - Vertical margin
  • .p-{1-5} - Padding all sides
